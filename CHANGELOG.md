## [0.1.0] - 2026-01-11

### Highlights

- Initial release of OmniRetrieve - a unified retrieval library for Go
- Support for Vector RAG, Graph RAG, and Hybrid retrieval strategies

### Added

- feat: Core retrieval interfaces (`Retriever`, `Query`, `Result`, `ContextItem`)
- feat: Vector retrieval with `Index`, `BatchIndex`, and `IndexManager` interfaces
- feat: Graph retrieval with `KnowledgeGraph` and `GraphManager` interfaces
- feat: Hybrid retrieval with configurable policies (`Parallel`, `VectorThenGraph`, `GraphThenVector`)
- feat: Observability support with span tracing (Phoenix, Opik, Langfuse compatible)
- feat: Reranking support (`CrossEncoder`, `Heuristic`, `Chain`)
- feat: In-memory implementations for testing (`VectorIndex`, `KnowledgeGraph`, `HashEmbedder`)
- feat(providers): pgvector provider with HNSW/IVFFlat index types
- feat(providers): Batch operations via COPY and multi-row INSERT
- feat(providers): JSONB metadata filtering support
